=============І. Описание задания=============================================================
Дана сущность Developer

Необходимо реализовать консольное приложение со следующим функционалом:

1. Создание разработчика
2. Редактирование
3. Получение данных (чтение)
4. Удаление

В качестве хранилища использовать текстовый файл

Необходимо реализовать следующие классы

Developer - POJO класс, который содержит данные о разработчике
Класс DeveloperDAO - содержит методы для работы с текстовым файлом
Класс DeveloperView - который содержит логику работы с консолью
Класс DeveloperController - экземпляр которого создаётся в DeveloperView для обработки запросов
из консоли и содержит DeveloperDAO.
Программа должна прекращать работу только по указанию пользователя
Выход из программы в результате исключения недопустим.

=============ІІ.Описание методов программы===================================================
 - Класс Developer содержит переменные для хранения данных о разработчике, конструкторы, геттеры
 и сеттеры, переопределенные методы toString(), equals() и hashCode().
 - Класс DeveloperDAO содержит методы для работы с файлом Developers.txt: save() - сохраняет
 разработчика в файл; remove() - сохраняет всё из файла в коллекцию, удаляет файл, удаляет
 искомого разработчика из коллекции, оставшиеся элементы коллекции записываются в новый файл
 с таким же именем; update() - удаляет искомого разработчика из файла и запсивыет его с изменениями;
 getByID() - возвращает разработчика из файла по ИД; getAll() - возвращает коллекцию разработчиков
 из файла.
 - Класс DeveloperController создает экземпляр класс DeveloperDAO и вызывает его методы по запросы
 от пользователя.
 - Класс DeveloperView создает экземпляр класса DeveloperController, выводит меню на экран и передает
 запросы пользователя в controller.
 - Класс DeveloperAppRuner создает экземпляп класса DeveloperView, являвется стартовой точкой программы.
